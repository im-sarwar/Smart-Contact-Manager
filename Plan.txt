Smart Contact Manager

Features:
Add New Contacts
View Contacts
Search Contacts
Edit/Update Contacts
Delete Contacts

Plan
First of all Main Menu will be displayed to the user:
	which will contain "Menu" and "Quit" Options Only.
In Menu all feature options will be provided to user and an option to go back to main menu and quit immediately.
Then for every feature a function will be created to do specified task, and an option will be given to go back to menu or exit immediately (will look into exiting immediately idea).

Design

Main Menu
Just Display the user options to go to menu and quit.


Functions of each Feature

Add New Contacts
	1. Contacts will be saved in dictionary, using key as name and value as number.
	2. data will stored in file contacts_book.json.
	3. Option to exit immediately will also be provided as in every function/feature.

View Contacts
	1. Contacts_book.json file will be opened, and converted back into dict or list, and contacts 	   will be shown to user.
	2. Option to exit immediately will also be provided as in every function/feature.

Search Contacts
	1. Contact_book.json file be accesed, converted back, read, and then user will have to enter 	the desired name/key, and value/contactwill be displayed for that value.
	2. Option to exit immediately will also be provided as in every function/feature.

Edit/Update Contacts
	1. In this contacts_book.json file will be accessed, the user will be asked to enter the 	contact/key he want to change, then insert the new number/value for key/contact.
	2. Option to exit immediately will also be provided as in every function/feature.

Delete Contacts
	1. In this contacts_book.json file will be accessed, the user will be asked to enter the 	contact/key he want to delete, then respective key/contact will be deleted.
	2. Option to exit immediately will also be provided as in every function/feature.
